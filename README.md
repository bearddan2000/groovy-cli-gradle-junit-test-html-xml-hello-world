# groovy-cli-gradle-junit-test-html-xml-hello-world

## Description
A POC for gradle app using JUnit.
Uses test task to dump html and xml
to shared folder.

## Tech stack
- groovy
- gradle
  - junit

## Docker stack
- gradle:jdk11

## To run
`sudo ./install.sh -u`

## To stop
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
